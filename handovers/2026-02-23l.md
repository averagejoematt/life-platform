## Bugs found and fixed during deployment
- **Cloudflare blocks Python urllib** — Habitify API behind Cloudflare rejects default `Python-urllib/3.13` User-Agent. Fixed: set `User-Agent: LifePlatform/1.0` in both test script and Lambda.
- **URL encoding** — `+` in `+00:00` timezone not URL-encoded caused 500 from Habitify `/journal` endpoint. Fixed: switched from manual query string to `urllib.parse.urlencode()`.
- **Hygiene typo** — Habitify area was "Hygeine"; Matthew fixed in app. (If API still returns typo, it may be cached.)
