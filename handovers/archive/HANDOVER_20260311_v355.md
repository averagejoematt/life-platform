## Key Technical Reminders

- **Deploy scripts MUST use `deploy/deploy_lambda.sh`** — never inline zip logic. The helper strips path prefix via temp dir. Inline `zip file.zip lambdas/file.py` buries module under `lambdas/` in the zip — Lambda can't find it.
- **CDK stack names:** LifePlatformCore, LifePlatformIngestion, LifePlatformCompute, LifePlatformEmail, LifePlatformOperational, LifePlatformMcp, LifePlatformWeb, LifePlatformMonitoring
- **CDK invocation:** `cd cdk && npx cdk deploy <StackName> --require-approval never` (node v25 warning is harmless)
