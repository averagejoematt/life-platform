## Key Learnings

### S3 Sync --delete Safety Rule
**Any `aws s3 sync site/ --delete` MUST exclude Lambda-generated files:**
```
--exclude "public_stats.json"
--exclude "config/experiment_library.json"
--exclude "config/challenges_catalog.json"
--exclude "data/character_stats.json"
```
These files exist only in S3, generated by Lambdas. Deleting `public_stats.json` breaks the entire homepage.

### Patch Idempotency
Running the same patch script twice caused duplications (CSS blocks, theme toggle buttons, accent borders). Future patches should check for existing markers before inserting.

### email-subscriber Lambda
- Name: `email-subscriber` (NOT `life-platform-email-subscriber`)
- Region: `us-west-2`
- NOT in deploy_lambda.sh's hardcoded list — deploy manually with `zip -j` + `aws lambda update-function-code`
