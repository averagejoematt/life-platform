# Brand marks

The AJM dial — from the Claude Design project "AJM Logo Marks", direction **3a**.

**Three files here are hand-authored masters. Everything else is generated.**

| Master | What it is |
|---|---|
| `mark-a.svg` | the single-"A" dial — used wherever the mark renders small (favicon, the 20px header mark) |
| `mark-ajm.svg` | the full AJM monogram — used at app-icon sizes and above |
| `lockup.svg` | monogram + "averagejoematt" wordmark, for social banners and profiles |

Masters use `currentColor` for ink and `class="ember"` on the one accent element,
so a single file can be tinted for either theme.

Generated files carry a `GENERATED by …` banner. Do not edit them — edit the
master and re-run:

```bash
python3 scripts/build_brand_assets.py           # rebuild
python3 scripts/build_brand_assets.py --check   # fail if anything drifted
```

That script also writes `site/favicon.ico`, `site/apple-touch-icon*.png` and
`site/assets/icons/icon-{192,512,maskable-512}.png`.

Two things it does that a hand export cannot: it converts the wordmark text to
outlines (an SVG used as `<img>`, a favicon, or by a social crawler loads no
webfont, so live `<text>` would fall back to the wrong monospace), and it bakes
per-theme colours, since `currentColor` does not resolve inside a CSS
`background-image`.
