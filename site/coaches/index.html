Under the hood

How It Works

  • 01 Computation. Five compute Lambdas run every morning before 11 AM, aggregating metrics from 26 data sources into daily snapshots, character scores, and adaptive mode signals.
  • 02 Orchestration. Each coach receives domain-specific data and the previous week's context. They see what the other coaches wrote, enabling cross-domain awareness without cross-domain noise.
  • 03 Generation. Coaches write their analyses in first person using Claude Opus, each constrained to their voice profile and analytical style. No generic summaries — every paragraph reflects a distinct perspective.
  • 04 State update. Predictions are tracked, confidence levels are recorded, and each coach's memory persists across weeks. When a coach is wrong, the system knows — and so do they.
