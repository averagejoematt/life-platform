Data flow

How data moves through the system

Every morning at 5 AM, data flows from 19 sources into a single DynamoDB table, gets processed by Lambda compute functions, and feeds Claude’s daily intelligence brief.

System design

Three-layer architecture

Everything flows inward to a single DynamoDB table, then outward through Claude. The design philosophy: get data in automatically, store it cheaply, make it queryable without a data engineering background.

Tech stack

What it's built on

Infrastructure
AWS Lambda + CDK
48 functions across 8 CDK stacks. All IAM roles dedicated and least-privilege. EventBridge scheduling.
DynamoDB + S3
Single-table design. On-demand billing. PITR. KMS CMK encryption. Static site on CloudFront.
SES + SNS + SQS
Email delivery, alert routing, dead-letter queue for failed Lambdas.
AI / Intelligence
Claude (Anthropic)
Sonnet for coaching, Haiku for classification. ~$3/month API cost. MCP protocol for tool calls.
MCP Server (custom)
95-tool Lambda backed by DynamoDB. OAuth 2.1 + HMAC auth. 31-module package. Nightly cache warmer.
Hypothesis Engine
Weekly Pearson correlation matrix (23 pairs, BH FDR correction) feeds AI hypothesis generation.
Quality + CI/CD
853 automated tests
CDK handler consistency, S3 path linting, IAM/secrets cross-check, MCP registry integrity, DynamoDB patterns.
GitHub Actions CI
Lint → test → plan → deploy (manual approval gate). OIDC federation. Auto-rollback on smoke failure.
Synthetic canary
Every 4 hours: DynamoDB round-trip, S3 round-trip, MCP endpoint health. SES alert on failure.
Intelligence layer

95 tools that know your body

The MCP server gives Claude direct access to every data point. Natural language queries across 18 months of longitudinal health data — no SQL, no dashboards, no context switching.

Ask anything. "Did my HRV drop the week after I increased Zone 2 training?" "What's the correlation between my sleep onset time and next-day recovery?" "Which habits most predict a good P40 score?"

The platform answers these questions because it has the data, the statistical methods, and 95 purpose-built tools to surface them.
