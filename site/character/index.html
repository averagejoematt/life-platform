Methodology

How scores are computed

Each pillar’s raw score (0–100) is computed nightly from real wearable, habit, and biometric data — not self-reported. Scores feed into an XP system that determines character level.

Pillar weights are configurable but currently equal (14.3% each). The composite Character Score is the weighted average of all 7 pillars. Level progression uses a logarithmic XP curve — early levels come fast, higher levels require sustained performance.
